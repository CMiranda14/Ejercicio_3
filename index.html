PROPIEDADES CSS

Propiedades Generales:

font-family: Define el la familia tipográfica.

color: Define el color del texto o de la tipografía.

width: Define el ancho de un elemento, y le valor se pued escribir en pixeles, ems, o porcentaje.

Max-height o min-heigth: Define el alto máximo o mínimo de un elemento.

font-size: Define el tamaño de la fuente y el valor se puede escribir en pixels o ems.

padding: Es la distancia desde el borde de un elemento hasta su contenido.

margin: Es la distancia entre un elemento y otro (desde el borde del elemento hacia afuera).

border: Define el borde de un elemento, su color, su estilo y grosor.

background: Define os fondos de un objeto.

Propiedades para texto

text-ident: Desplazamiento de la primera línea del texto. Sus valores son longitud y porcentaje.

text-aling: Alineamiento del texto. Sus valores son: left, right, center y justify.

text-decoration: Efectos de subrayado, tachado. Sus valores son: none, underline, overline, linethrough.

letter-spacing: Espacio entre caracteres. Sus valores son: normal y longitud.

word-spacing: Espacio entre palabras. Sus valores son: normal y longitud.

text-transform: Transoformación a mayúsculas y minusculas. Sus valoores son: capitalize, uppercase, lowercase, none.

line-height: Da un tamaño del espacio entre lineas.

vertical-aling: Alineación vertical. Sus valores son: top, middle, bottom, baseline, sub, super, valor.

Propiedades para Cajas

width: Define el ancho de un elemento, y le valor se pued escribir en pixeles, ems, o porcentaje.

height:especifica la altura del area de contenido de un elemento.

padding: Es la distancia desde el borde de un elemento hasta su contenido.

margin: Es la distancia entre un elemento y otro (desde el borde del elemento hacia afuera).

border: Define el borde de un elemento, su color, su estilo y grosor.

Conección css a html

Colocamos la etiqueta <link>, luego colocamos dentro de la etiqueta un atributo "rel" y dentro de ese rel entre "", colocamos la palabra "stylesheet". Luego colocamos otro atributo junto a "rel", el cual es "href" y dentro de ese atributo colocamos el nombre de nuestro css. Ejemplo:

<link rel="stylesheet"href="css/design.css">.
